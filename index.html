Billing & Parsing Guidelines

This portal supports two parsing modes. Local Offline Mode (No API Key Required): If your bills are named with the amount (e.g. zomato_345.pdf or uber_120.png), details are parsed instantly offline. Gemini AI Mode (API Key Required): If filenames are generic (e.g. zomato.pdf or invoice.png), a Gemini API Key is required to read the bill contents using Gemini AI.
