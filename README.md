# 📊 Claim Statement Portal

A premium, glassmorphic web portal built for **Pratishthan Software Ventures Pvt Ltd** employees to easily prepare and export expense claim statements.

## 🚀 Live Demo / Local Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local Vite server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

---

## ☁️ Deployment Instructions

Since this is a modern static frontend app, you can deploy it to production in seconds:

### 1. Deploy to Vercel (Recommended)
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project folder and follow the prompts.
3. Your app will be live on a production URL in less than 30 seconds!

### 2. Deploy to Netlify
1. Connect your Github repository to [Netlify](https://www.netlify.com/).
2. Set the build settings:
   - **Build Command**: `npm run build`
   - **Publish Directory**: `dist`
3. Click **Deploy**!

### 3. Deploy to GitHub Pages
1. Build the production files: `npm run build`
2. Deploy the generated `dist` folder to your `gh-pages` branch.
